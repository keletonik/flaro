## Mandatory sync protocol (run before ANY push)

```bash
# 1. Fetch the latest remote main.
git fetch origin main

# 2. Rebase your local work on top — never reset, never force.
git pull --rebase origin main

# 3. If the rebase pauses on a conflict, STOP and hand control back to the
#    user. Do NOT auto-resolve by keeping only "ours" — you will delete the
#    motion layer. Conflicts usually mean both sides touched the same lines.

# 4. Verify these files exist on disk BEFORE you push:
test -f artifacts/aide/src/lib/motion.ts                   || { echo "MISSING motion.ts — ABORT"; exit 1; }
test -f artifacts/aide/src/components/ui/CountUp.tsx       || { echo "MISSING CountUp.tsx — ABORT"; exit 1; }
test -f artifacts/aide/src/components/ui/StreamCursor.tsx  || { echo "MISSING StreamCursor.tsx — ABORT"; exit 1; }
test -f artifacts/aide/src/components/ui/Reveal.tsx        || { echo "MISSING Reveal.tsx — ABORT"; exit 1; }
grep -q "aide-stream-cursor" artifacts/aide/src/index.css  || { echo "MOTION LAYER MISSING FROM index.css — ABORT"; exit 1; }
grep -q "CountUp"       artifacts/aide/src/pages/dashboard.tsx || { echo "dashboard.tsx reverted — ABORT"; exit 1; }
grep -q "drawerRight"   artifacts/aide/src/components/AidePA.tsx || { echo "AidePA.tsx reverted — ABORT"; exit 1; }

# 5. Typecheck + build.
pnpm --filter @workspace/aide run typecheck
pnpm --filter @workspace/aide run build

# 6. Only now push.
git push origin main
```
